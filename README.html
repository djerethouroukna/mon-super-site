📚 Documentation du Projet

🎯 À Propos

C'est un portfolio professionnel moderne et réactif pour un développeur web. Le site est construit avec une architecture modulaire et des bonnes pratiques de développement.

📁 Structure du Projet

mon-super-site/
├── index.html              # Page d'accueil
├── about.html              # Page À propos
├── services.html           # Page Services & Tarifs
├── portfolio.html          # Galerie de projets
├── contact.html            # Formulaire de contact
├── Apropos.html            # Alias pour about.html
├── Contact.html            # Alias pour contact.html (majuscule)
│
├── css/
│   ├── main.css            # Point d'entrée CSS
│   ├── variables.css       # Variables et design system
│   ├── reset.css           # Normalisation CSS
│   ├── layout.css          # Système de grille et layout
│   ├── header.css          # En-tête et navigation
│   ├── footer.css          # Pied de page
│   ├── components.css      # Composants réutilisables
│   └── pages.css           # Styles spécifiques aux pages
│
├── js/
│   ├── navigation.js       # Gestion du menu mobile
│   ├── forms.js            # Validation et gestion des formulaires
│   └── utils.js            # Fonctions utilitaires
│
└── assets/
    └── images/             # Dossier pour les images

Typographie

✨ Fonctionnalités

  • Design moderne et professionnel
  • Entièrement responsive (mobile-first)
  • Navigation fluide avec menu hamburger
  • Formulaire de contact avec validation
  • Système de grille CSS flexible
  • Animations fluides et transitions
  • Code bien organisé et documenté
  • Performance optimisée

🚀 Comment Utiliser

  1. Ouvrez index.html dans votre navigateur
  2. Explorez les différentes pages via la navigation
  3. Testez le formulaire de contact
  4. Vérifiez la responsivité en redimensionnant la fenêtre

📱 Breakpoints Responsive

  • Desktop: 1200px+
  • Tablet: 768px - 1199px
  • Mobile: Moins de 768px
  • Small Mobile: Moins de 480px

🔧 Personnalisation

Pour personnaliser le design :

  1. Modifiez les couleurs dans css/variables.css
  2. Changez la typographie dans css/reset.css
  3. Mettez à jour le contenu dans les fichiers HTML
  4. Remplacez les images dans le dossier assets/images
